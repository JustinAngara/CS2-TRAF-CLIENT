## Features
- Aimbot (aim -> fire architectur rather than shoot -> aim)
- Autofire (if crosshair is aiming at head shoot)
- No Recoil
- Bhop
- ESP (name, health, skeleton, bars, colors)
- FoV Custom Croshair
- Cool title at the top
